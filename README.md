# CodeAlpha_Language_Translation_Tool 🌐🔐

A secure, responsive, and clean **Language Translation Tool** developed during the CodeAlpha Artificial Intelligence Internship.

## ✨ Features

 🔐 Secure Login & Register: Dynamic custom account creation with a local secure password verification layer.
 🌐 Real-Time Translation: Dual-language conversion supporting global languages like English, Urdu, Arabic, Spanish, etc.
 🎛️ Advanced Voice Controls: Built-in Text-to-Speech (TTS) with real-time speed and pitch adjusting sliders.
 🛡️ Smart Guest Mode: Local session limits that allow 5 free translations before prompting for account setup.
 🌓 UI/UX Themes: Smooth Dark and Light mode toggle with an active translation history stack.

## 🛠️ Tech Stack Used

* Frontend: HTML5, CSS3 (Custom Variables & Blur Effects)
* Logic: Vanilla JavaScript (ES6+)
* API Integration: MyMemory Translation API
